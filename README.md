# aops-dist

Distribution artifacts for [academicOps](https://github.com/nicsuzor/academicOps) plugins.

## About

This repository contains auto-generated distribution builds. **Do not edit files here directly.**

All source code lives in the main [academicOps](https://github.com/nicsuzor/academicOps) repository.

## Contents

- `aops-core-claude/` - Claude Code plugin
- `aops-core-gemini/` - Gemini CLI plugin  
- `antigravity/` - Antigravity extension
- `*.tar.gz`, `*.zip` - Compressed distribution archives

## Updates

This repository is automatically updated by GitHub Actions when changes are pushed to academicOps main branch. Each commit message references the source commit SHA.
